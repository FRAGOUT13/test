# Basic Movement Unreal C++ Example

This directory contains a minimal Unreal Engine C++ project that demonstrates basic character movement.

## Features

- `ABasicMovementCharacter` that supports WASD/arrow key movement and camera control.
- `ABasicMovementGameMode` sets the custom character as the default pawn.
- Example build scripts (`.Target.cs` files) and module rules.

## Usage

1. Copy the `BasicMovement` folder into your Unreal Engine projects directory.
2. Generate project files using the Unreal Engine editor or `GenerateProjectFiles` script.
3. Open the project in Unreal Engine and run.

This project is meant as a starting point for learning how to implement movement in Unreal C++.
